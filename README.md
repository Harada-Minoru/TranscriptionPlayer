# TranscriptionPlayer
Sound Player for Transcription.

①セットアップ方法
・ZipファイルをGitHubからダウンロード
・Zipファイルを展開
・TranscriptionPlayer.exe を起動

※起動の際に、以下のような不明の発行先に関する警告がでることがありますが、
※稀に非公式のソフトということで、セキュリティソフトに引っかかってしまうことがあります。
